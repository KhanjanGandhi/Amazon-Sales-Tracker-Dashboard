# 📦 Amazon Sales Analysis Dashboard

## 📌 Project Overview
The Amazon Sales Analysis Dashboard is an interactive Power BI dashboard designed to analyze sales performance, order fulfillment, courier status, product categories, and shipment distribution. It provides valuable business insights through visualizations and key performance indicators (KPIs) to support data-driven decision-making.

## 🎯 Objectives
- Analyze overall sales performance.
- Monitor order and shipment status.
- Identify top-selling product categories.
- Compare delivery partner performance.
- Analyze fulfillment methods.
- Track state-wise shipment distribution.
- Support business decision-making with actionable insights.

## 📊 Dashboard Features
- Total Sales Amount KPI
- Courier Status Analysis
- Top Categories Performance
- Delivery Partner Comparison
- Fulfillment Analysis
- Size-wise Order Distribution
- State-wise Shipment Analysis
- Interactive Date Filters and Slicers

## 🛠️ Tools & Technologies
- Power BI
- Microsoft Excel / CSV Dataset
- Data Modeling
- Data Visualization
- DAX
- Power Query

## 📈 Key Metrics
- Total Sales Amount: 11.58M
- Product Categories Analysis
